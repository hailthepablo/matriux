# Matriux
Matriux is a Python program I made to help me with linear algebra, since doing row reductions on paper kinda sucks. To use it, you will need to download my custom libraries from this link:

[https://github.com/hailthepablo/loganslibraries]

and you will also need to install the Colorama module by running
```bash
pip install colorama
```
After that you should be all set.
